## 阅读准备

这篇指南基于 Minecraft 1.18.2 版本，并基于 Forge 40.1.19，和 Java 17.0.3。这篇指南假设读者拥有一定基于 Java 17 的开发经验，一定量的 Minecraft 游玩经验，和足够通畅的网络环境。这篇指南同时假设读者拥有一台能够正常运行 Minecraft 1.18.x，并拥有至少 4 GB 空闲内存的计算机。
